⚙ Setup & Configuration

Follow these steps once to connect this app to your Google Sheet. Takes about 10 minutes.

1

Create your Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it "Monnai Finance Tasks".

Create these tabs (click the + at the bottom): Tasks DB, Dashboard, Sprint View, Team Registry

2

Set up the Tasks DB tab headers

In the Tasks DB tab, paste these headers in Row 1 (A1 through K1):

Task ID Task Name Project Assignee Region Start Date Due Date Status Area Vendor Involved Priority Notes
3

Add the Apps Script to enable writing

In your Google Sheet, click Extensions → Apps Script. Delete any existing code and paste all of the following, then click Save (💾):

4

Deploy the Apps Script as a Web App

  • Click Deploy → New deployment
  • Set type to Web app
  • Execute as: Me
  • Who has access: Anyone (or "Anyone with Google account" if preferred)
  • Click Deploy, authorize when prompted
  • Copy the Web App URL (looks like: https://script.google.com/macros/s/ABC.../exec)
5

Share your sheet so this app can read it

  • In the Google Sheet, click the Share button (top right)
  • Under "General access", change it to "Anyone with the link" and set permission to Viewer
  • Click Copy link — this is the URL you'll paste in Step 6 below

The URL looks like: https://docs.google.com/spreadsheets/d/1aBcD.../edit?usp=sharing — just paste the whole thing, the app will extract what it needs.

8

Dashboard formulas for your Google Sheet

Copy these formulas into the Dashboard tab of your Google Sheet:

-- Column order: A=Task ID, B=Task Name, C=Project, D=Assignee, E=Region, F=Start Date, G=Due Date, H=Status, I=Area, J=Vendor, K=Priority, L=Notes -- Cell B2: Total Tasks =COUNTA('Tasks DB'!A2:A) -- Cell B3: Not Started =COUNTIF('Tasks DB'!H2:H,"Not Started") -- Cell B4: In Progress =COUNTIF('Tasks DB'!H2:H,"In Progress") -- Cell B5: Done =COUNTIF('Tasks DB'!H2:H,"Done") -- Cell B6: Blocked =COUNTIF('Tasks DB'!H2:H,"Blocked") -- Cell B7: Overdue (past due date, not Done) =COUNTIFS('Tasks DB'!G2:G,"<"&TODAY(),'Tasks DB'!H2:H,"<>Done",'Tasks DB'!G2:G,"<>") -- Cell B8: High Priority =COUNTIF('Tasks DB'!K2:K,"High") -- Tasks by Assignee (put in D2, drag down for each person) =COUNTIF('Tasks DB'!D2:D, C2) -- Tasks by Area (put in G2, drag down) =COUNTIF('Tasks DB'!I2:I, F2) -- Tasks by Region (put in J2, drag down) =COUNTIF('Tasks DB'!E2:E, I2) -- Sprint View tab: Current sprint tasks (not Done, High or Medium priority) =FILTER('Tasks DB'!A2:L, ('Tasks DB'!H2:H<>"Done")*('Tasks DB'!K2:K<>"Low"))
9

Host this HTML file (free, no coding)

Option A — Simplest: Just open this file in Chrome. It works fully offline. Share the file with Cindy via Google Drive or email.

Option B — GitHub Pages (free, always accessible online):

  • Go to github.com → Create free account → New repository (name: finance-tasks, set to Public)
  • Click "Add file → Upload files" → drag this HTML file in
  • Go to repository Settings → Pages → Source: main branch → Save
  • Your app will be live at: https://yourusername.github.io/finance-tasks/finance-task-manager.html
